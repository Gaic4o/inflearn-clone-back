# npm 라이브러리 도구


- aws-sdk : SDK 를 사용하면 AWS 에서 S3, EC2 등을 제공하므로 복잡한 코드 사용 하지 않아도 됩니다. 

- axios : 프론트 부분이랑 데이터 주고 받을 떄 사용. 
  
- bcrypt : 비밀번호 암호화 할 떄 사용 합니다.
  
- helmet : 
  
- hpp : HTTP Paramter Pollution (보안 상 쓰는 건데 기억 안남 나중 정리)
  
- morgan : express 가 동작하면서 나오는 메시지들을 콘솔에 출력. 
  
- multer : 이미지 파일을 업로드 할 떄 쉽게 이미지를 업로드 할 수 있음. 
  
- multer-s3 : s3 를 사용하면 amazon S3에 업로드 가능 (굳이 서버에다가 이미지 파일들을 용량 차지 않게 할 수 있음)
  
- mysql2 : promise 지원 해 주므로 다른 모듈 x 사용이 가능. 
    (기존 mysql 를 사용했는 데 mysql은 callback 기반이기 떄문에 
    promise를 사용하지 x 따로 promise-mysql 모듈을 따로 설치해야 함)

- passport : 
  
- passport-local : 내부 local 직접 로그인 가능. 
    (kakao 라면 passport-kakao)
  
- pm2 : 나중 배포 할 떄 사용. 
  
- sequelize : 데이터 다대다 관계 편이 
  
- sequelize-cli : - config, models, migrations, s.. 
    -> sequelize db:create 을 통해  db 생성.
